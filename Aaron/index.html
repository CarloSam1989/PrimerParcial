Variables en Python

Una variable es una forma de identificar, de forma sencilla, un dato que se encuentra almacenado en la memoria del ordenador, el cuál, puede cambiar durante el flujo del programa. Una variable nos permite acceder fácilmente a dicho dato para ser manipulado y transformado.

Asignar un valor a una variable

para asignar un valor (un dato) a una variable se utiliza el operador de asignación =.

Modificar el valor de una variable

Para modificar el valor de una variable en Python, basta con asignarle un nuevo valor en cualquier momento y lugar después de la definición.
